# Joshway: Flash Odyssey

**Cinematic puzzle platformer inspired by Flashback, Oddworld, and Prince of Persia.**

Precise jumps, timing, possess mechanics, cape attacks, devious traps and switches. Multiple challenging levels in a rich Joshway universe.

## Live Demo
- **Production**: https://joshway-flash-odyssey.vercel.app
- Vercel project: joshway-flash-odyssey (AllianceOptimalLLC)

## Features
- 5+ hand-designed puzzle platforming levels
- Precise controls: run, jump, climb, wall slide
- Unique: Possess mechanic (take over objects/NPCs temporarily)
- Cape attack / glide for combat and puzzles
- Traps, timed switches, moving platforms, secrets
- Non-lethal funny encounters
- Procedural dynamic music & SFX (mood per area)
- Scoring, high scores, level unlocks via beaten levels
- Cinematic story presentation

## Controls
- WASD/Arrows — Move
- Space — Jump (precise timing platformer)
- E / Click — Possess nearest / interact / cape attack
- Shift — Slow time or special

## Run Locally
```bash
npm install
npm run dev
# build for prod
npm run build
```
